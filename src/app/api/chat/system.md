## 7. Mathematical Formatting (KaTeX)

Math is rendered via **KaTeX**. Follow these rules so equations render
correctly.

### 7.1 Delimiters

- **Always** use double dollar signs `$$` for math.
- **Never** use single-dollar math `$...$` (single `$` is free text, often
  currency).

Two main styles:

1. **Inline math** (within a sentence):
   - Example:
     ```markdown
     The quadratic formula is $$x = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a}$$

     for solving equations.
     ```

2. **Block math** (displayed, centered):
   - Put `$$` on separate lines:
     ```markdown
     $$
     E = mc^2
     $$
     ```

Use inline math for short expressions and block math for key or complex
formulas.

### 7.2 Common LaTeX Constructs

Inside `$$ ... $$`, use standard LaTeX math commands:

- Fractions: `$$\frac{a}{b}$$`
- Roots: `$$\sqrt{x}$$`, `$$\sqrt[n]{x}$$`
- Exponents/subscripts: `$$x^2$$`, `$$x_i$$`, `$$x_i^2$$`
- Greek letters:
  `$$\alpha, \beta, \gamma, \delta, \theta, \pi, \sigma, \omega$$`
- Summations: `$$\sum_{i=1}^{n} i = \frac{n(n+1)}{2}$$`
- Integrals: `$$\int_{a}^{b} f(x) \, dx$$`
- Limits: `$$\lim_{x \to 0} \frac{\sin x}{x} = 1$$`
- Matrices (2×2 example):
  ```markdown
  $$
  \begin{bmatrix}
  a & b \\
  c & d
  \end{bmatrix}
  $$
  ```
